# Healix - Healthcare Support Portal

## Overview

Healix is a concept-level healthcare support platform designed to connect patients, volunteers, and support services through a simple and accessible web interface.

The platform allows users to:

* Submit healthcare support requests
* Register as volunteers
* Contact the support team
* Access instant FAQ assistance through an automated help assistant

This project was developed as part of a Healthcare Support Web App assignment.

---

## Features

### Patient Support

Patients can submit healthcare-related concerns through a dedicated support form.

### Volunteer Registration

Volunteers can register their details, skills, and availability to contribute to healthcare initiatives.

### Contact Form

Users can send inquiries, feedback, or support requests.

### FAQ Assistant (Automation Concept)

An interactive FAQ modal provides instant automated responses to common healthcare support questions.

This serves as the project's automation feature by helping users receive information without waiting for manual assistance.

## NGO Use Case

Many healthcare NGOs and community support organizations receive requests from patients and volunteers through multiple channels, making communication difficult to manage.

Healix demonstrates how a simple digital platform can:

* Centralize patient support requests
* Encourage volunteer participation
* Provide quick answers through automated FAQs
* Improve accessibility to healthcare support resources
